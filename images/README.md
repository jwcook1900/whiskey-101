# Images

Add Whiskey's hero photo here as **`whiskey.jpg`**. The hero will show a 🐕
emoji placeholder until the file is present, then display the photo
automatically (cropped to a circle, focused on his face).

- Recommended: a roughly square or portrait JPG, at least 600×600px.
- The crop focus is set in `styles.css` via `.hero__img { object-position }`
  — tweak it if his face isn't centred.
